# 日程表

一款通过 Vibe Coding 方式搭建的任务管理工具原型。

## 技术工具
- OpenAI Codex（代码生成与补全）

## 核心功能
- 任务增删改查
- 优先级标记（高/中/低）
- 多视图切换（月视图/周视图）

## 项目背景
本项目是我在探索 AI 辅助编程过程中的实践产物。通过 Vibe Coding 方式，从零开始搭建了一款可交互的 Web 应用，完整经历了“需求描述 → Codex 生成初版 → 本地运行测试 → 错误日志反馈 → 定向修改 Prompt → 迭代优化”的闭环工作流。

## 运行方式
在浏览器中直接打开 `index.html` 文件即可运行。

## 个人收获
- 深刻理解了大模型输出需要“人类校验与结构化评测”；
- 建立了一套可与 AI 协作的最小化产品开发 SOP；
- 通过实际调试前端渲染与状态管理问题，强化了问题拆解能力（展示层/数据层分层定位）。
